🚀 QUICK START GUIDE:

STEP 1: Backup File Lama
────────────────────────
- Simpan backup dari index.html, main.css, main.js
- Jangan hapus file lama sampai testing selesai

STEP 2: Merge CSS
────────────────
- Copy semua isi dari additional_styles.css
- Paste di akhir file main.css Anda
- Atau buat file terpisah dan import di HTML

STEP 3: Update HTML
──────────────────
- Replace index.html dengan index_updated.html
- Atau copy section #education dan #contact dari file baru

STEP 4: Setup Email Service
───────────────────────────
Pilih salah satu:

OPTION A: Formspree (Paling Mudah)
  1. Go to https://formspree.io → Sign up
  2. Create new form, copy FORM_ID
  3. Di HTML, cari: 'https://formspree.io/f/YOUR_FORM_ID'
  4. Ganti YOUR_FORM_ID dengan Form ID Anda
  5. Done! Test langsung

OPTION B: EmailJS (Recommended)
  1. Go to https://www.emailjs.com → Sign up
  2. Setup email service (Gmail, Outlook, dll)
  3. Create email template
  4. Copy PUBLIC_KEY, SERVICE_ID, TEMPLATE_ID
  5. Update di HTML sesuai instructinos
  6. Test

STEP 5: Testing
───────────────
- Buka website di browser
- Test kirim message via form
- Check email inbox (atau spam folder)
- Verifikasi message terkirim dengan benar

STEP 6: Deploy
──────────────
- Push changes ke github
- Deploy ke hosting (Vercel, Netlify, etc)
- Test ulang di live site

⚠️ IMPORTANT NOTES:

1. Email Service
   - Tidak bisa send email langsung dari HTML/JS tanpa backend
   - Harus gunakan service seperti Formspree, EmailJS, atau backend sendiri
   - Formspree paling simple untuk pemula

2. Form Validation
   - Client-side validation sudah include (required attributes)
   - EmailJS/Formspree handle server-side validation

3. Browser Compatibility
   - Modern browsers (Chrome, Firefox, Safari, Edge)
   - IE tidak support (async/await)

4. CORS
   - Formspree dan EmailJS sudah handle CORS
   - Backend API perlu setup CORS headers

5. Rate Limiting
   - Formspree: 50 submissions/month (free)
   - EmailJS: 200 emails/month (free)
   - Backend: unlimited (tergantung email provider)

❓ FAQ:

Q: Bagaimana cara test form di localhost?
A: Formspree dan EmailJS support localhost. Gmail SMTP perlu app password.

Q: Berapa lama email terkirim?
A: Biasanya instant (< 1 detik). Check spam folder jika tidak ada di inbox.

Q: Apa yang bisa diisi di form?
A: Name, Email, Message. Bisa tambah field lain dengan edit HTML/JS.

Q: Gimana jika tidak mau pakai service?
A: Buat backend sendiri dengan Node.js + Express + Nodemailer. Lihat guide.

Q: Apakah email saya aman?
A: Formspree dan EmailJS sudah trusted. Jangan share API keys public.

Q: Bisa customize email template?
A: Ya, dengan EmailJS bisa custom template. Formspree auto format.
